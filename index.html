Reading this ledger: "Overall odds" is the published chance of winning any prize — including free tickets and breakeven prizes — averaged across every ticket printed for that game. It does not account for prizes already claimed, and it isn't in the daily "prizes claimed" report, so most games here start with it blank until you add it from the ticket back, a photo, or ask Claude to look up a specific game. "Top prizes remaining %" is a separate, faster-updating signal — a rough proxy for whether a game is "played out" — but it doesn't recalculate true odds, since it doesn't account for how many tickets are still unsold. Thumbnails are pulled directly from Texas Lottery's own ticket images by game number; if one fails to load, the game number is still shown so you can spot it in the store. Neither odds signal changes your chances on any individual ticket, and past results never affect future draws. For live official numbers, check texaslottery.com directly. If scratch tickets stop feeling like entertainment, Texas offers free, confidential help through the Council on Problem and Compulsive Gambling of Texas (24/7: 1-800-522-4700).
